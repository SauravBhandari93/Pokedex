**Pokedex** is a project made with learning purposes.

[Demo](https://pokdexpokemon.netlify.app/)

## Getting started

```sh
git clone git@github.com:SauravBhandari93/Pokedex.git
cd Pokedex
npm install
npm start # open localhost:3000
```
